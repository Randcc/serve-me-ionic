# serve-me-ionic

> Build with [Ionic](https://ionicframework.com/), version 4

## How to use

```shell script
# after clone
cd serve-me-ionic
npm install
# open in browser
ionic serve
# if you have not install ionic
npm install -g @ionic/cli
# if you have not cordova
npm i -g cordova
#if you want to build android apk
ionic cordova build android --prod
```
